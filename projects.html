branschdag-2023-03

Branschdag-23-03 är en app som jag gjort för att göra det enklare att hitta företagen på Nackademins branschdag-evenemang. Du kan klicka på klassrummen för att se vilka företag som finns där, filtrera efter ditt ämne för att bara se företag som är intressanta för dig och klicka på företagens namn för att se deras information.

https://henrikvilhelmberglund.com/branschdag-2023-03/
https://github.com/henrikvilhelmberglund/branschdag-2023-03

branschdag-2023-10

Branschdag-23-10 är en uppdaterad version av Branschdag-23-03. Den viktigaste nya funktionen är att istället för att kopiera och klistra in data skrev jag ett scraper-skript som använder Puppeteer för att hämta data från eventwebbplatsen och exportera data till .js-filer vilket gör uppdatering/kontroll av datat mycket enklare.

https://henrikvilhelmberglund.com/branschdag-2023-10/
https://github.com/henrikvilhelmberglund/branschdag-2023-10
